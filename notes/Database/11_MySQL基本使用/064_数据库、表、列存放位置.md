# 数据库、表、列存放位置
    1. 查看数据库存放路径
+ mysql> show variables like 'datadir';
+ ![img_1104.png](https://raw.githubusercontent.com/Oumu33/notes/main/notes/images/img_1104.png)
+ #ls -l /var/lib/mysql （数据库对应磁盘的目录）
+ ![img_4112.png](https://raw.githubusercontent.com/Oumu33/notes/main/notes/images/img_4112.png)
    1. 查看表存放路径
+ mysql>  show tables from mysql like 'user';
+ [root@offcn  ~]# ls -l /var/lib/mysql/mysql/user*
+ mysql库的user表对应/var/lib/mysql/mysql/user.frm,user.MYD,user.MYI
+ 
    1. 查看列存放位置
+ mysql>  desc mysql.user;
+ [root@offcn  ~]# strings /var/lib/mysql/mysql/user.frm
+ user表中的列存放在var/lib/mysql/mysql/user.frm中
+ 
    1. 查看行存放位置
+ mysql>  select user,host,authentication_string from mysql.user;
+ [root@offcn  ~]# strings /var/lib/mysql/mysql/user.MYD
+ user表中的行可以在/var/lib/mysql/mysql/user.MYD中找到一些内容
+ ![img_2256.png](https://raw.githubusercontent.com/Oumu33/notes/main/notes/images/img_2256.png)
    1. 数据库结构与文件结构关系
+ 数据库1（目录1）
+ 表  （文件）
+ 列1，列2，列3，……(  一个表至少有一列 )
+ 行1，行2，行3，……（一个表可以有0到N行）
+ 数据库2（目录2）
+ 表1
+ 表2
+ 表3
+ ……
+ 数据库3（目录3）
+ 表1
+ ……
