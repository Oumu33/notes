# 元字符
| 字符 | 含义 |
| --- | --- |
| <font style="color:rgb(51, 51, 51);">\b</font> | <font style="color:rgb(51, 51, 51);">匹配一个单词边界，也就是指单词和空格间的位置</font> |
| <font style="color:rgb(51, 51, 51);background-color:rgb(246, 244, 240);">\B</font> | <font style="color:rgb(51, 51, 51);background-color:rgb(246, 244, 240);">匹配非单词边界</font> |
| <font style="color:rgb(51, 51, 51);background-color:rgb(246, 244, 240);">\d</font> | <font style="color:rgb(51, 51, 51);background-color:rgb(246, 244, 240);">匹配一个数字字符。等价于 [0-9]。</font> |
| <font style="color:rgb(51, 51, 51);">\D</font> | <font style="color:rgb(51, 51, 51);">匹配一个非数字字符。等价于 [^0-9]。</font> |
| <font style="color:rgb(51, 51, 51);">\n</font> | <font style="color:rgb(51, 51, 51);">匹配一个换行符</font> |
| <font style="color:rgb(51, 51, 51);">\s</font> | <font style="color:rgb(51, 51, 51);">匹配任何空白字符，包括空格、制表符、换页符等等。等价于 [ \f\n\r\t\v]。</font> |
| <font style="color:rgb(51, 51, 51);background-color:rgb(246, 244, 240);">\S</font> | <font style="color:rgb(51, 51, 51);background-color:rgb(246, 244, 240);">匹配任何非空白字符。等价于 [^ \f\n\r\t\v]。</font> |
| <font style="color:rgb(51, 51, 51);">\w</font> | <font style="color:rgb(51, 51, 51);">匹配字母、数字、下划线。等价于'[A-Za-z0-9_]'。</font> |
| <font style="color:rgb(51, 51, 51);background-color:rgb(246, 244, 240);">\W</font> | <font style="color:rgb(51, 51, 51);background-color:rgb(246, 244, 240);">匹配非字母、数字、下划线。等价于 '[^A-Za-z0-9_]'。</font> |
