# 帮助命令
# 一、获得帮助信息
1. 命令名称：man
2. 命令英文原意：manual
3. 命令所在路径：/usr/bin/man
4. 执行权限：所有用户
5. 语法：man[命令或配置文件]
6. 功能描述：获得帮助信息
7. 范例：

①$man Is

（查看is命令的帮助信息）

②$man services

（查看配置文件services的帮助信）

# 二、获得Shell内置命令的帮助信息
1. 命令名称：help
2. 命令所在路径：Shell内置命令
3. 执行权限：所有用户
4. 语法：help命令
5. 功能描述：获得Shell内置命令的帮助信息
6. 范例：

$help umask

（查看umask命令的帮助信息）
