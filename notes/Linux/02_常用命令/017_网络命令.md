# 测试网络连通性
1. 命令名称：**ping**
2. 命令所在路径：/bin/ping
3. 执行权限：所有用户
4. 语法：ping 选项IP地址 -c 指定发送次数
5. 功能描述：测试网络连通性
6. 范例：# ping 192.168.1.23

# 查看和设置网卡信息
1. 命令名称：**ifconfig**
2. 命令英文原意：interface configure
3. 命令所在路径：/sbin/ifconfig
4. 执行权限：root
5. 语法：ifconfig 网卡名称 IP地址
6. 功能描述：查看和设置网卡信息
7. 范例：# ifconfig eth0       192.168.8.223

# 查看发送电子邮件
1. 命令名称：**mail**
2. 命令所在路径：/bin/mail
3. 执行权限：所有用户
4. 语法：mail[用户名]
5. 功能描述：查看发送电子邮件
6. 范例：# mail root

# 列出目前与过去登入系统的用户信息
1. 命令名称：**last**
2. 命令所在路径：/usr/bin/last
3. 执行权限：所有用户
4. 语法：last
5. 功能描述：列出目前与过去登入系统的用户信息
6. 范例：#last

# 检查某特定用户上次登录的时间
1. 命令名称：**lastlog**
2. 命令所在路径：/usr/bin/lastlog
3. 执行权限：所有用户
4. 语法：lastlog
5. 功能描述：检查某特定用户上次登录的时间
6. 范例：①#lastlog②#lastlog -u 502

# 显示数据包到主机间的路径
1. 命令名称：**traceroute**
2. 命令所在路径：/bin/traceroute
3. 执行权限：所有用户
4. 语法：traceroute
5. 功能描述：显示数据包到主机间的路径
6. 范例：#traceroute [www.lampbrother.net](http://www.lampbrother.net)

# 显示网络相关信息
1. 命令名称：**netstat**
2. 命令所在路径：/bin/netstat
3. 执行权限：所有用户
4. 语法：netstat [选项]
5. 功能描述：显示网络相关信息
6. 选项

| -t | tcp协议 |
| --- | --- |
| -u | udp协议 |
| -l | 监听 |
| -r | 路由 |
| -n | 显示ip地址和端口号 |


7. 范例

```bash
# 查看本机监听的端口
netstat -tlun
# 查看本机所有的网络连接
netstat -an
# 查看本机路山表
netstat -rn
```
