# 条件判断
# 一、按照文件类型进行判断
| 测试选项  | 作 用 |
| --- | --- |
| -b 文件 | 判断该文件是否存在，并且是否为<font style="color:red;">块设备</font>文件（是块设备文件为真） |
| -c文件 | 判断该文件是否存在，并且是否为<font style="color:red;">字符设备</font>文件（是字符设备文件为真） |
| -d 文件  | 判断该文件是否存在，并且是否为<font style="color:red;">目录</font>文件（是目录为真） |
| -e 文件  | 判断该文件是否存在（存在为真） |
| -f 文件  | 判断该文件是否存在，并且是否为<font style="color:red;">普通</font>文件（是普通文件为真） |
| -L 文件 | 判断该文件是否存在，并且是否为<font style="color:red;">符号链接</font>文件（是符号链接文件为真） |
| -p 文件  | 判断该文件是否存在，并且是否为<font style="color:red;">管道</font>文件（是管道文件为真） |
| -s 文件  | 判断该文件是否存在，并且是否为<font style="color:red;">非空</font>（非空为真） |
| -S 文件 | 判断该文件是否存在，并且是否为<font style="color:red;">套接字</font>文件（是套接字文件为真） |


1. 两种判断格式

![](https://via.placeholder.com/800x600?text=Image+8fac40d09dc0e600)

[ -d /root ] && echo "yes" || echo "no"

1. 第一个判断命令如果正确执行，则打印“yes”，否则打印“no”

# 二、按照文件权限进行判断
| 测试选项 | 作 用 |
| --- | --- |
| -r 文件 | 判断该文件是否存在，并且是否该文件拥有<font style="color:red;">读</font>权限（有读权限为真） |
| -w文件 | 判断该文件是否存在，并且是否该文件拥有<font style="color:red;">写</font>权限（有写权限为真） |
| -x 文件 | 判断该文件是否存在，并且是否该文件拥有<font style="color:red;">执行</font>权限（有执行权限为真） |
| -u 文件 | 判断该文件是否存在，并且是否该文件拥有<font style="color:red;">SUID</font>权限（有SUID权限为真） |
| -g 文件 | 判断该文件是否存在，并且是否该文件拥有<font style="color:red;">SGID</font>权限（有SGID权限为真） |
| -k 文件 | 判断该文件是否存在，并且是否该文件拥有<font style="color:red;">SBit</font>权限（有SBit权限为真） |

# 四、两个整数之间比较
| 测试选项 | 作 用 |
| --- | --- |
| 整数1 -eq 整数2 | 判断整数1是否和整数2<font style="color:red;">相等</font>（相等为真） |
| 整数1 -ne 整数2 | 判断整数1是否和整数2<font style="color:red;">不相等</font>（不相等位置） |
| 整数1 -gt 整数2 | 判断整数1是否<font style="color:red;">大于</font>整数2（大于为真） |
| 整数1 -lt 整数2 | 判断整数1是否<font style="color:red;">小于</font>整数2（小于位置） |
| 整数1 -ge 整数2 | 判断整数1是否<font style="color:red;">大于等于</font>整数2（大于等于为真） |
| 整数1 -le 整数2 | 判断整数1是否<font style="color:red;">小于等于</font>整数2（小于等于为真） |

# 六、多重条件判断
| 测试选项 | 作 用 |
| --- | --- |
| 判断1 -a 判断2 | <font style="color:red;">逻辑与</font>，判断1和判断2都成立，最终的结果才为真 |
| 判断1 -o 判断2 | <font style="color:red;">逻辑或</font>，判断1和判断2有一个成立，最终的结果就为真 |
| ！判断 | <font style="color:red;">逻辑非</font>，使原始的判断式取反 |
