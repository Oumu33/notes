# 三、free命令查看内存使用状态
1. [root@localhost ~]#      free [-b|-k|-m|-g]
2. 选项：

| -b：  | 以字节为单位显示 |
| --- | --- |
| -k： | 以KB为单位显示，默认就是以KB为单位显示 |
| -m： | 以MB为单位显示 |
| -g： | 以GB为单位显示 |


![](https://via.placeholder.com/800x600?text=Image+1b2cc7d012c736ca)

1. 缓存和缓冲的区别
+  简单来说缓存（cache）是用来加速数据从硬盘中“读取”的
+ 缓冲（buffer）是用来加速数据“写入”硬盘的。
