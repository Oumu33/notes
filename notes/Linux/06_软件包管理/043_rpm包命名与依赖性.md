# rpm包命名与依赖性
# 一、rpm包命名原则
例如：httpd-2.2.15-15.e16.centos.1.i686.rpm

| httpd | 软件包名 |
| --- | --- |
| 2.2.15 | 软件版本 |
| 15 | 软件发布的次数 |
| e16.centos | 适合的Linux平台 |
| i686 | 适合的硬件平台 |
| rpm | rpm包扩展名 |


# 二、rpm包依赖性
1. 树形依赖：a→b→c
2. 环形依赖：a→b→c→a
3. 模块依赖：模块依赖查询网站：www.rpmfind.net
