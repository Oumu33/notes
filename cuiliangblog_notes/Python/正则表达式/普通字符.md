# 普通字符
| 字符 | 描述 |
| --- | --- |
| **<font style="color:rgb(51, 51, 51);background-color:rgb(236, 234, 230);">[ABC]</font>** | <font style="color:rgb(51, 51, 51);">匹配 </font>**<font style="color:rgb(51, 51, 51);background-color:rgb(236, 234, 230);">[...]</font>**<font style="color:rgb(51, 51, 51);"> 中的所有字符</font> |
| **<font style="color:rgb(51, 51, 51);background-color:rgb(236, 234, 230);">[^ABC]</font>** | <font style="color:rgb(51, 51, 51);background-color:rgb(246, 244, 240);">匹配除了 </font>**<font style="color:rgb(51, 51, 51);background-color:rgb(236, 234, 230);">[...]</font>**<font style="color:rgb(51, 51, 51);background-color:rgb(246, 244, 240);"> 中字符的所有字符</font> |
| **<font style="color:rgb(51, 51, 51);background-color:rgb(236, 234, 230);">[A-Z]</font>** | <font style="color:rgb(51, 51, 51);">[A-Z] 表示一个区间，匹配所有大写字母，[a-z] 表示所有小写字母。</font> |
| **<font style="color:rgb(51, 51, 51);background-color:rgb(236, 234, 230);">.</font>** | <font style="color:rgb(51, 51, 51);background-color:rgb(246, 244, 240);">匹配除换行符（\n、\r）之外的任何单个字符，相等于 [^\n\r]。</font> |
