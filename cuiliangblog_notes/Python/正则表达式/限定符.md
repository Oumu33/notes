# 限定符

> 分类: Python > 正则表达式
> 更新时间: 2026-01-10T23:34:27.044361+08:00

---

| 符号 | 功能 |
| --- | --- |
| **<font style="color:rgb(51, 51, 51);background-color:rgb(236, 234, 230);">?</font>** | <font style="color:rgb(51, 51, 51);">前面的字符最多只可以出现一次（0次或1次）。</font> |
| **<font style="color:rgb(51, 51, 51);background-color:rgb(236, 234, 230);">*</font>** | <font style="color:rgb(51, 51, 51);">前面的字符可以不出现，也可以出现一次或者多次（0次、或1次、或多次）。</font> |
| **<font style="color:rgb(51, 51, 51);background-color:rgb(236, 234, 230);">+</font>** | <font style="color:rgb(51, 51, 51);">前面的字符必须至少出现一次（1次或多次）。</font> |
| <font style="color:rgb(51, 51, 51);">{n}</font> | <font style="color:rgb(51, 51, 51);background-color:rgb(246, 244, 240);">匹配确定的 n 次</font> |
| <font style="color:rgb(51, 51, 51);">{n,}</font> | <font style="color:rgb(51, 51, 51);">至少匹配n 次</font> |
| <font style="color:rgb(51, 51, 51);background-color:rgb(246, 244, 240);">{n,m}</font> | <font style="color:rgb(51, 51, 51);background-color:rgb(246, 244, 240);">n <= m。最少匹配 n 次且最多匹配 m 次</font> |
