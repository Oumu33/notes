# 导入和导出容器

> 分类: Docker > 容器
> 更新时间: 2026-01-10T23:35:05.351026+08:00

---

1. 导出容器
+ 导出容器是指导出一个已经创建的容器到一个文件，不管此时该容器是否处于运行状态  
[root[@docker ](/docker ) ~]# docker export -o centos.tar f0761a1df372   
[root[@docker ](/docker ) ~]# docker export -o  f0761a1df372 > centos.tar 
2. 导入文件成为镜像  
[root[@docker ](/docker ) ~]# docker import centos.tar centos:test 
+ docker import和docker load区别在于：load导入镜像存储文件，import导入容器快照文件。容器快照文件仅保存当前快照状态，丢弃所有历史记录和元数据信息，镜像存储文件将保存完整记录，体积更大。从容器快照文件导入时可以重新制定标签等元数据信息。
