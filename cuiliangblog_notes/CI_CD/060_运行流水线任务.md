# 添加流水线文件
在gitlab仓库中项目根目录添加一个.gitlab-ci.yml文件，文件内容如下

```yaml
stages:
  - build
  - deploy

build:
  stage: build
  script:
    - echo "start build"

deploy:
  stage: deploy
  script:
    - echo "start deploy"
```

这个流水线共包含两个job，分别是build 和 deploy。

buildjob包含一个stage build，运行构建命令。

deployjob包含一个stage deploy，运行部署命令。
