# 关机重启命令
# 一、shutdown关机命令
1. 命令格式：[root@localhost~]#shutdown      [选项] 时间
2. 选项：



| -c | 取消前一个关机命令 |
| --- | --- |
| -h | 关机 |
| -r | 重启 |




# 二、其他关机命令
1. [root@localhost~]#halt
2. [root@localhost~]#poweroff
3. [root@localhost~]#init 0

# 三、其他重启命令
1. [root@localhost~]#reboot
2. [root@localhost~]#init 6

# 四、系统运行级别


| 0 | 关机 |
| --- | --- |
| 1 | 单用户（类似windows安全模式） |
| 2 | 不完全多用户，不含nfs服务 |
| 3 | 完全多用户 |
| 4 | 未分配 |
| 5 | 图形界面 |
| 6 | 重启 |


```bash
# 修改系统默认运行级别
[root@localhost~]#cat /etc/inittab
id:3:initdefault
# 查询系统运行级别
[root@localhost~]#runlevel
```

# 五、退出登录命令
1. [root@localhost      ~ ] # logout
