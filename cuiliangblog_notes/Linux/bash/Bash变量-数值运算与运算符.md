# 一、declare声明变量类型
[root@localhost ~]# declare [+/-][选项]变量名

1. 选项：

| -： | 给变量设定类型属性 |
| --- | --- |
| +：  | 取消变量的类型属性 |
| i： | 将变量声明为整数型（integer） |
| --x： | 将变量声明为环境变量 |
| -p： | 显示指定变量的被声明的类型 |

# 三、运算符
| 优先级  | 运算符  | 说明 |
| --- | --- | --- |
| 13 | -, +  | 单目负、单目正 |
| 12  | !, ~  | 逻辑非、按位取反或补码 |
| 11  | * , / , %  | 乘、除、取模 |
| 10  | +, -  | 加、减 |
| 9  | << ,   >>  | 按位左移、按位右移 |
| 8  | < =,   > =, < , >  | 小于或等于、大于或等于、小于、大于 |
| 7  | == , !=  | 等于、不等于 |
| 6  | &  | 按位与 |
| 5  | ^  | 按位异或 |
| 4  | |  | 按位或 |
| 3  | &&    | 逻辑与 |
| 2  | ||  | 逻辑或 |
| 1 | =,+=,-   =,*=,/=,%=,&=, ^=,|=, <<=, >>=  | 赋值、运算且赋值 |
