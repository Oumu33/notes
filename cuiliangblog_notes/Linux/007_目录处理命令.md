# 一、创建目录
命令名称：**mkdir**

命令英文原意：make directories

命令所在路径：/bin/mkdir

执行权限：所有用户

语法：mkdir-p[目录名]

功能描述：①创建新目录②-p递归创建

范例：$mkdir       -p/tmp/Japan/boduo②$mkdir /tmp/Japan/longze/tmp/Japan/cangjing

# 二、切换目录
命令名称：**cd**

命令英文原意：change directory

命令所在路径：shell内置命令

执行权限：所有用户

语法：cd[目录]

功能描述：切换目录

范例：

```bash
# 切换到指定目录
cd /tmp/Japan/boduo
# 回到上一级目录
cd ..
# 回到家目录
cd
# 回到上次工作的目录
cd -
```

# 三、目录处理命令
命令名称：**pwd**

命令英文原意：print working       directory

命令所在路径：/bm/pwd

执行权限：所有用户

语法：pwd

功能描述：显示当前目录

范例：①$pwd /tmp/Japan

# 四、删除空目录命令
命令名称：**rmdir**

命令英文原意：remove empty       directories

命令所在路径：/bm/nndir

执行权限：所有用户

语法：rmdir[目录名]

功能描述：删除空目录

范例：$rmdir       /tmp/Japan/b0duo

# 五、复制目录命令
命令名称：**cp**

命令英文原意：copy

命令所在路径：/bin/cp

执行权限：所有用户

语法：cp①-rp[原文件或目录][目标目录]②-r复制目录③-p保留文件属性

功能描述：复制文件或目录

范例：①$cp       -r/tmp/Japan/cang11nghoot（将目录/tmp/Japan/cang11nghoot复制到目录/root下）②$cp -rp/tmp/Japan/boduo /tmp/Japan/longze       /root（将/tmp/Japan目录下的boduo和longze目录复制到/root下，保持目录属性）

# 六、剪切文件、改名命令
命令名称：**mv**

命令英文原意：move

命令所在路径：/bin/mv

执行权限：所有用户

语法：mv[原文件或目录][目标目录]

功能描述：剪切文件、改名

# 七、删除目录或文件
命令名称：**rm**

命令英文原意：remove

命令所在路径：/bin/rm

执行权限：所有用户

语法：rm①-rf[文件或目录]②-r删除目录③-f强制执行

功能描述：删除文件

范例：①$rm /tmp/yum.log（删除文件/tmp/ymm]og）②$rm-rf       /tmp/Japan/longze（删除目录/tmp/Japan/longze）
