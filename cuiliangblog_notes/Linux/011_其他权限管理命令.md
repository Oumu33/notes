# 一、改变文件或目录的所有者
1. 命令名称：chown
2. 命令英文原意：change file ownership
3. 命令所在路径：/bin/chown
4. 执行权限：所有用户
5. 语法：chown[用户][文件或目录]
6. 功能描述：改变文件或目录的所有者
7. 范例：$ chown shenchao      fengjie

（改变文件fengjie的所有者为shenchao）

# 二、改变文件或目录的所属组
1. 命令名称：chgrp
2. 命令英文原意：change file group      ownership
3. 命令所在路径：/bin/chgrp
4. 执行权限：所有用户
5. 语法：chgrp[用户组][文件或目录]
6. 功能描述：改变文件或目录的所属组
7. 范例：$chgrp lampbrother      fengjie

（改变文件fenglie的所属组为lampbrother）

# 三、显示、设置文件的默认权限
1. 命令名称：umask
2. 命令英文原意：the user      file-creation mask
3. 命令所在路径：Shell内置命令
4. 执行权限：所有用户

 

1. 语法：umask[S] 

-s 以rwx形式显示新建文件缺省权限

1. 功能描述：显示、设置文件的缺省权限
2. 范例：$ umask -S
