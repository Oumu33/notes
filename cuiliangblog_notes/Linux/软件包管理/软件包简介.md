# 软件包简介
# 一、软件包分类
1. 源码包（脚本安装包）
2. 二进制包（rpm包、系统默认包）

# 二、源码包
1. 源码包的优点：
+ 开源，如果有足够的能力，可以修改源代码
+ 可以自由选择所需的功能
+ 软件是编译安装，所以更加适合自己的系统，更加稳定也效率更高
+ 卸载方便
2. 源码包的缺点
+ 安装过程步骤较多，尤其安装较大的软件集合时（如LAMP环境搭建），容易出现拼写错误
+ 编译过程时间较长，安装比二进制安装时间长
+ 因为是编译安装，安装过程中一旦报错新手很难解决

# 三、rpm包
1. 二进制包的优点
+ 包管理系统简单，只通过几个命令就可以实现包的安装、升级、查询和卸载
+ 安装速度比源码包安装快的多
2. 二进制包缺点：
+ 经过编译，不再可以看到源代码
+ 功能选择不如源码包灵活
+ 依赖性
