# 一、source命令

或

[root@localhost ~]# . 配置文件

# 二、环境变量配置文件简介
1. 环境变量配置文件中主要是定义对系统的操作环境生效的系统默认环境变量，比如PATH、HISTSIZE、PS1、HOSTNAME等默认环境变量。
+ /etc/profile
+ /etc/profile.d/*.sh
+ ~/.bash_profile
+ ~/.bashrc
+ /etc/bashrc
