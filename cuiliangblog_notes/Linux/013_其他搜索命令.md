# 其他搜索命令
| 功能 | 命令 | 选项 |
| --- | --- | --- |
| **在文件资料库中查找文件** | **locate** |   |
| **搜索命令所在目录及别名** | **which** |   |
| **搜索命令所在目录及帮助文档路径** | **whereis** |   |
| **在文件中搜寻字串匹配的行并输出** | **grep** | **-i****不区分大小写**<br/>**-v****排除指定字串** |


# 一、在文件资料库中查找文件
1. 命令名称：**locate**
2. 命令所在路径：/usr/bin/locate
3. 执行权限：所有用户
4. 语法：locate 文件名
5. 功能描述：在文件资料库中查找文件
6. 范例：$ locate inittab

# 二、搜索命令所在目录及别名
1. 命令名称：**which**
2. 命令所在路径：/usr/bin/which
3. 执行权限：所有用户
4. 语法：which命令
5. 功能描述：搜索命令所在目录及别名信息
6. 范例：$whichIs

# 三、 搜索命令所在目录及帮助文档路径
1. 命令名称：**whereis**
2. 命令所在路径：/usr/bin/whereis
3. 执行权限：所有用户
4. 语法：whereis[命令名称]
5. 功能描述：搜索命令所在目录及帮助文档路径
6. 范例：$whereisIs

# 四、在文件中搜寻字串匹配的行并输出
1. 命令名称：**grep**
2. 命令所在路径：/bin/grep
3. 执行权限：所有用户
4. 语法：grep-iv[指定字串][文件]

| -i | 不区分大小写 |
| --- | --- |
| -v | 排除指定字串 |


1. 功能描述：在文件中搜寻字串匹配的行并输出
2. 范例：#grep mysql      /root/install.log
