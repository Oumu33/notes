# 链接命令
# 一、生成链接文件
1. 命令名称：In
2. 命令英文原意：link
3. 命令所在路径：/bin/ln
4. 执行权限：所有用户
5. 语法：In [原文件][目标文件]

| -s | 创建软链接 |
| --- | --- |


1. 功能描述：生成链接文件
2. 范例：

①$ In -s /etc/issue /tmp/issue.soft（创建文件/etc/issue的软链接 /tmp/issue.soft）

②$ In /etc/issue /tmp/issuehard（创建文件/etc/issue的硬链接/tmp/issue.hard）

# 二、软链接特征（类似windows快捷方式）
1. lrwxrwxrwx
+  l：软链接 
+ rwxrwxrwx：所有软链接文件权限
+ 文件大小——只是符号链接
+ /tmp/issue.soft->/etc/issue  箭头指向原文件

# 三、硬链接特征（类似保存文件属性拷贝）
1. 拷贝cp-p并同步更新两个文件信息
2. 可通过i节点识别
3. 不能跨分区
4. 不能针对目录使用
