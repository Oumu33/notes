# rpm服务的管理-独立服务管理
![](https://via.placeholder.com/800x600?text=Image+16c6302ad2a4360c)

# 一、RPM包安装服务的位置
1. RPM安装服务和源码包安装服务的区别就是安装位置的不同
+ 源码包安装在指定位置，一般是/usr/local/
+  RPM包安装在默认位置中
2.  /etc/init.d/：启动脚本位置
3.  /etc/sysconfig/：初始化环境配置文件位置
4.  /etc/：配置文件位置
5.  /etc/xinetd.conf：xinetd配置文件
6.  /etc/xinetd.d/：基于xinetd服务的启动脚本
7.  /var/lib/：服务产生的数据放在这里
8.  /var/log/：日志

# 二、独立服务的启动
1. /etc/init.d/独立服务名 start|stop|status|restart|
2.  service 独立服务名 start|stop|restart||status

# 三、独立服务的自启动
1.  chkconfig [--level 运行级别] [独立服务名] [on|off]
2.  修改/etc/rc.d/rc.local文件
3.  使用ntsysv命令管理自启动
