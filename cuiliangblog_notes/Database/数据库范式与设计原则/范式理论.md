# 一、关系型数据库的特点
1. 基本组成单位为二维表
2. 各二维表之间存在一定的关系
+ 数据为什么不能都放在一张表里面？

![](../../images/img_1209.png)

**一、第一范式**

1. 如果关系模式R的每个关系r的属性都是不可分的数据项，那么就称R是第一范式的模式
2. 主要特点是实体的属性不能再分，即列不能再分
3. 错误设计
