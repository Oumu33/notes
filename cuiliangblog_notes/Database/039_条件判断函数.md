# 一、if(expr,v1,v2)
1. 如果表达式expr成立，返回v1值；否则，返回v2值
2. 如果学生高考分大于等于520，其为统招生，否则其为委培生，从学生表中查找，显示姓名，考分，类型（统招或委培）

mysql> select sname,score,if(score>=520,'统招','委培') 类型 from stu;
