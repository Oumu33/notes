# 一、创建集合
1. 语法格式：

db.createCollection(name, options)

1. 参数说明：

name: 要创建的集合名称

options: 可选参数, 指定有关内存大小及索引的选项

1. options 可以是如下参数：

| **字段** | **类型** | **描述** |
| --- | --- | --- |
| capped | 布尔 | （可选）如果为   true，则创建固定集合。固定集合是指有着固定大小的集合，当达到最大值时，它会自动覆盖最早的文档。<br/>**当该值为 true 时，必须指定 size 参数。** |
| autoIndexId | 布尔 | （可选）如为   true，自动在 _id 字段创建索引。默认为 false。 |
| size | 数值 | （可选）为固定集合指定一个最大值，以千字节计（KB）。<br/>**如果 capped 为 true，也需要指定该字段。** |
| max | 数值 | （可选）指定固定集合中包含文档的最大数量。 |


1. 示例

> db.createCollection("123")

{ "ok" : 1 }

1. 在 MongoDB 中，你不需要创建集合。当你插入一些文档时，MongoDB 会自动创建集合。

> db.mycol2.insert({"name" : "菜鸟教程"})

WriteResult({ "nInserted" : 1 })

> show tables

mycol2

1. 创建固定集合 mycol，整个集合空间大小 6142800      KB, 文档最大个数为 10000 个。

> db.createCollection("mycol", { capped : true, autoIndexId : true, size : 

   6142800, max : 10000 } )

{ "ok" : 1 }

# 三、删除集合
1. 语法格式：

db.collection.drop()

1. 返回值

如果成功删除选定集合，则 drop() 方法返回 true，否则返回 false。

1. 示例
+ 查看集合

> show collections

123

mycol2

+ 删除mycol2集合

> db.mycol2.drop()

true

+ 查看集合

> show collections

123
