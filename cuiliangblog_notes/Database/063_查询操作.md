    1. 当前用户可以访问的数据库
+ mysql> show databases;
    1. 查看MySQL服务器版本
+ mysql> select version();
    1. 查看当前连接的用户
+ mysql> select user();
    1. 查看当前服务器日期和时间
+ mysql> select now();
    1. 查看“mysql”数据库的表
+ mysql> show tables from mysql;
+ 或者
+ mysql>use  mysql;
+ mysql>  show tables;
    1. 查看当前使用的数据库
+ mysql> select database();
    1. 查看mysql库的user表的列
+ mysql>  desc mysql.user;
+ 如果当前正在使用mysql库
+ mysql>desc  user;
    1. 查看mysql库的user表的行
+ mysql> select * from mysql.user;
+ user表的列太多，导致折行，把列转成行来看，这样清晰一些
+ mysql>  select * from mysql.user\G
