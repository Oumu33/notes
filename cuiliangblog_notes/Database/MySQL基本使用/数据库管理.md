# 数据库管理

> 分类: Database > MySQL基本使用
> 更新时间: 2026-01-10T23:34:18.058514+08:00

---

    1. 创建数据库
+ mysql>create  database test;
+ mysql>show  databases;
+ [root@offcn  ~]# ls -l /var/lib/mysql
+ [root@offcn  ~]# ls -l /var/lib/mysql/test
+ [root@offcn  ~]# ls -l /var/lib/mysql/test/db.opt
    - create database test       character set 'utf8' collate 'utf8_general_ci';
+ 创建名为  “test”的数据库，使用utf8编码可以存储中文
    - character set 
+ 指定字符串以何种编码存储。如果在库中建表，没有指定以何种编码存储，默认使用utf8编码存储表中的字符串
    - collate
+ 定义字符串排序的顺序和比较的方式
+ utf8_general_ci，字符串按照UTF8编码比较大小，不区分大小写
+ ci  = case insensitive
    1. 删除数据库
+ mysql>drop  database test;
+ [root@offcn  ~]# ls -l /var/lib/mysql
+ 目录会被删除
+ 如果里面有表文件也会被删除
+ 千万不要删除mysql,sys,performance_schema等这些系统数据库
+ 如果删除了，mysql服务器程序崩溃，切记危险！
