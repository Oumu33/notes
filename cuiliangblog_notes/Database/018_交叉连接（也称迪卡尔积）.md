# 交叉连接（也称迪卡尔积）

> 来源: Database
> 创建时间: 2021-02-12T00:25:21+08:00

---

笛卡尔连接或交叉连接从两个或多个连接表返回笛卡尔乘积的记录。因此，它相当于一个内部联接，其中联接条件始终计算为真或者联接条件是从语句中空缺。

**语法**

笛卡尔连接的基本语法如下：

select * from a cross join b;

select * from a join b;
