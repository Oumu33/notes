| 数据类型 | 使用场景 |
| --- | --- |
| percent(0-100) | CPU、内存使用率 |
| bytes | 总内存大小 |
| seconds（s） | 系统运行时间 |
| bytes/sec | 磁盘读取速率 |
| bits/sec | 下行带宽速率 |
