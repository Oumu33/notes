# 生成打包报告
> 在项目打包完成后，如果生成的文件过大，会导致用户访问项目时间过长，需要对生成项目文件优化时，通过打包报告，可以直观的发现项目中存在的问题。
>

1. 打包项目

`vue-cli-service build --report`

2. 查看打包报告

    ![](https://via.placeholder.com/800x600?text=Image+96b484d9011d893c)

+ 打包报告会详细列出各个组件占用文件大小的详细信息，便于优化项目

![](https://via.placeholder.com/800x600?text=Image+0d7be5a993f73bdc)
